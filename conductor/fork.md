# Fork Attribution Plan

## Objective
Update the project to properly attribute the original creator by adding a fork notice to the README and committing the change.

## Proposed Solution
1. **Modify `README.md`**:
   - Add a notice directly under the main title (`# Tab Out`):
     ```markdown
     > **Note:** This repository is a fork of the original [Tab Out by @zarazhangrui](https://github.com/zarazhangrui/tab-out).
     ```
   - Keep the original "Built by Zara" credit intact at the bottom.

2. **Commit the Change**:
   - Stage the `README.md` file.
   - Create a commit with the message: `chore: add fork attribution to README`.

## Verification
- Review the `README.md` to ensure the attribution is clear.
- Verify the commit was successfully created in the local Git history.
